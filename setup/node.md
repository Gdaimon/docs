# Node

## Windows

```
Install-Module -Name power-nvm
nvm install latest
nvm default latest
```

## Unix

```
curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.33.8/install.sh | bash
souce ~/.nvm
nvm install --lts
nvm alias default node
```
